# 🚀 Quick Start Guide - Design Critique Dungeon 2.0

## ⚡ Get Started in 60 Seconds

### Step 1: Choose Your Mode

You have **3 ways** to experience the app:

#### Option A: Dungeon Mode (DOS RPG) ⚔️
```bash
open index-dungeon.html
```
**Best for:** Designers who love retro games and fantasy aesthetics

#### Option B: Carbon Mode (Professional) 💎
```bash
open index-carbon.html
```
**Best for:** Executives, stakeholders, and professional presentations

#### Option C: Demo Mode (Testing) 🎮
```bash
open demo-quest-system.html
```
**Best for:** Testing the quest system without uploading files

---

### Step 2: Upload a Design

1. Click **"Choose Design Scroll"** (Dungeon) or **"Select File"** (Carbon)
2. Select any UI screenshot from your computer
3. Preview appears automatically

---

### Step 3: Analyze

1. Click **"Enter the Dungeon"** (Dungeon) or **"Analyze Design"** (Carbon)
2. Wait 3 seconds for analysis
3. See results with:
   - Overall score
   - Detailed findings
   - Generated quests
   - XP reward

---

### Step 4: Complete Quests

1. Check the **Quest Log** (right panel in Dungeon, sidebar in Carbon)
2. See your active quests
3. Improve your design based on feedback
4. Re-analyze to update quest progress
5. Complete quests for XP rewards

---

### Step 5: Level Up

1. Watch your **XP bar** fill up
2. See your **Level** increase
3. Unlock new **Titles**
4. Track your **Statistics**

---

## 🎮 Demo Mode Controls

The demo has **4 interactive buttons**:

1. **🎲 Simulate Design Analysis**
   - Generates 3-7 random quests
   - Awards 50 XP
   - Updates character sheet

2. **✨ Simulate Design Improvement**
   - Increases quest progress by 25%
   - Updates progress bars
   - Completes quests at 100%

3. **✅ Complete Random Quest**
   - Instantly completes one quest
   - Awards full XP
   - Shows completion notification

4. **🔄 Reset All Progress**
   - Clears all quests
   - Resets XP to 0
   - Starts fresh

## 🎯 Common Tasks

### How to Switch Modes
**From Dungeon to Carbon:**
- Click **"Switch to Carbon Mode"** button in character sheet

**From Carbon to Dungeon:**
- Click **"Dungeon Mode"** button in header

### How to Reset Progress
**Option 1:** Use demo mode's reset button
**Option 2:** Clear browser localStorage:
```javascript
localStorage.clear();
location.reload();
```

### How to Export Results
- Click **"Export Report"** button (Carbon mode)
- Feature coming soon in Dungeon mode

### How to View Statistics
- **Dungeon Mode:** Check character sheet (left panel)
- **Carbon Mode:** Check progress card (right sidebar)
- **Demo Mode:** Character sheet shows all stats

## 🐛 Troubleshooting

### Quest Log is Empty
**Solution:** Complete an analysis first to generate quests

### XP Not Updating
**Solution:** Refresh the page - data persists in localStorage

### Image Won't Upload
**Solution:** Ensure file is an image (JPG, PNG, GIF, etc.)

### Analysis Stuck on Loading
**Solution:** 
- Check if using Demo Mode (no API key needed)
- Verify API key if using real AI provider
- Refresh page and try again

### Mode Switch Not Working
**Solution:** Manually navigate to the other HTML file
