⚡ Demo Mode Information

What is Demo Mode?

Carbon Quest currently runs in Demo Mode with sophisticated client-side analysis. This means the app analyzes your designs entirely in your browser—no server required, no data sent anywhere.

What It Analyzes Right Now:

  • IBM Color Palette - Compares pixels to 19 IBM Design Language colors
  • Carbon 16px Grid - Validates alignment with Carbon's 2x grid system
  • WCAG Contrast - Measures accessibility using luminance ratios
  • Design Type Detection - Identifies IBM UI vs photos vs games
  • Compliance Scoring - Calculates clarity, accessibility, and risk scores

This Is NOT a Mock Demo:

The analysis is real and functional. It samples pixels, measures distances, validates grids, and calculates scores using actual algorithms based on IBM Design Language, Carbon Design System, and WCAG standards.

Future: AI-Enhanced Analysis

Production mode could add AI-powered features like:

  • Natural language critiques (IBM watsonx.ai / Granite models)
  • Component identification and validation
  • Typography verification (IBM Plex detection)
  • Advanced layout analysis

Why Demo Mode?

Demo Mode lets stakeholders experience the complete workflow—upload, analyze, get feedback, unlock rewards, export findings—without requiring API keys, backend services, or paid AI models. The current analysis is already sophisticated enough for real design reviews.
